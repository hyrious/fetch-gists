## Fetch Gists

Ruby script to fetch my gist snippets and view them locally.

### Prerequisites

```shell
gem install http midori
```

### Clone All Gists

```shell
ruby pull-gists.rb
```

### Start a HTTP Server to View Gists

```shell
ruby gists.rb
```
